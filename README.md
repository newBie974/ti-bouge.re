# tiboug-web

Site vitrine de **Ti Boug — Bazar Péi**, l'app iOS contre la vie chère sur les fruits & légumes à La Réunion. Promesse : « Sak i lé bon marché, kan, é ousa ».

Landing statique servie sur [tiboug.re](https://tiboug.re).

## Stack
Astro (statique), CSS natif, déploiement GitHub Pages via GitHub Actions.

## Développement
```bash
npm install
npm run dev      # serveur local
npm run build    # build statique -> dist/
npm run preview  # prévisualise le build
```

Dépôt indépendant (pas de monorepo). Données de prix : mercuriales DAAF Réunion, licence etalab 2.0.
